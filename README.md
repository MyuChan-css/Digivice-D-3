
  # Diseñar Digivice D-3

  This is a code bundle for Diseñar Digivice D-3. The original project is available at https://www.figma.com/design/SPtOkVMqa4LNeUbOUkK5J1/Dise%C3%B1ar-Digivice-D-3.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
